# geometric-variance

Calculate geometric variance.

## Example

``` javascript
var geovar = require('geometric-variance');

geovar(0.5);
// => 2
```

## Installation

``` bash
$ npm install geometric-variance
```

## API

``` javascript
var geovar = require('geometric-variance');
```

### `geovar(p)`

Returns _Number_ geometric distribution variance of _Number_ success probability
`p`.
